# Astro Eleven - Live Zodiac Love-Match Clock

This project is a lightweight, GitHub Pages-friendly static web page built with HTML5, CSS3, JavaScript, and a tiny local jQuery-compatible helper.

## Structure

- index.html
- css/style.css
- js/zodiac-clock.js
- js/jquery.min.js
- images/astro-eleven-logo.png

## Run locally

Use any local static server, for example:

```bash
python -m http.server 8000
```

Then open http://127.0.0.1:8000/.

## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Astro Eleven Zodiac Clock"
git branch -M main
git remote add origin https://github.com/USERNAME/astro-eleven.git
git push -u origin main
```

Then enable Pages from the repository settings and serve the main branch.
